# reef-tools

**Modular Python package for reef and water quality analysis.**

Shared utilities for the DETSI group, covering data I/O, statistical analysis,
visualization, climate model handling, and water quality metrics.

## Package Structure

| Module | Purpose |
|--------|---------|
| [`reef_tools.io`](modules/io.md) | Data I/O — CSV helpers, netCDF loading |
| [`reef_tools.stats`](modules/stats.md) | Statistical analysis — trends, distributions |
| [`reef_tools.viz`](modules/viz.md) | Visualization — time series, faceted plots |
| [`reef_tools.climate`](modules/climate.md) | Climate model utilities — name parsing, labels |
| [`reef_tools.water_quality`](modules/water_quality.md) | Water quality metrics — DIN, TSS, discharge |
| [`reef_tools.utils`](modules/utils.md) | General utilities — decorators, helpers |
